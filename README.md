# ai-researcharea 🧰

> **This is a live workspace, not a finished project.** Everything here is actively being researched, built, and iterated on. Expect rough edges, works in progress, and things that change frequently. That's the point.

A public record of my journey exploring AI technologies — what I'm building, what I'm learning, and the tools and scripts I'm developing along the way. Covering Claude, OpenAI, Ollama (local LLMs), and wherever the research takes me next.

---

## What this is

I'm using this repo as a personal knowledge base and script library as I dig deeper into AI tools and workflows. Rather than keeping it private until it's "ready", I'm building it in the open. If something here is useful to you, great. If you spot something wrong or have suggestions, feel free to open an issue.

## Areas of research

### 🤖 Claude — [/claude](./claude/)
Scripts, tools, and notes built around Anthropic's Claude. Includes automation workflows, Notion integrations, and Claude-powered desktop apps.

### 🌐 OpenAI — [/openai](./openai/)
Exploration of OpenAI's API and tooling. Prompt libraries, scripts, and experiments.

### 🦙 Ollama — [/ollama](./ollama/)
Running and working with local LLMs via Ollama on a self-hosted Ubuntu machine. Notes on setup, model comparisons, and local AI workflows.
